Projet Discord Publique

Ici je posterai des projets sur lesquels je travaille ou j’ai travaillé dans le cadre des Bot Discord en JavaScript.

Comment faire une commande ?

C'est très simple il suffit de faire un devis en pdf, ce devis doit contenir:

  • Le type de bot: modération, statistique ou autres
  • Sur combien de serveurs pourrait être le bot (un ou plus)
  • Le détail des commandes qu'il devra effectuer
  • Le token du bot (je ne prends pas le risque de prendre mon propre bot pour des problèmes de permissions)
  • Une invitation vers un serveur où je pourrais contacter quelqu'un en cas de question sur ce dernier (pour éviter de perdre du temps et avoir des réponses trop lentes)
  • Votre désir d'un serveur tout fait pour vous avec un bot personnalisé [Payant]
  • Votre budget (merci de le préciser même s'il est nul)

Une fois le devis prêt veuillez me l'envoyer sur mon serveur [ici] dans un ticket.
Après la lecture de votre devis, je vous enverrai un message pour vous dire si j'accepte ou non de faire votre commande.
Je me donne le droit de refuser toute commande que je juge trop abusive ou ne respectant pas les ToS Discord

Comment payer ou faire un don ?

Tout paiement ou don se fera via PayPal.
Si c'est une commande payante ou qu'un don a été effectué la commande ne sera pas posté sur Github, dans ce cas-ci seul vous aurez le fichier de votre bot.
Dans le cas échéant, pour une commande gratuite, les fichier du bot sera publié sur GitHub.
Les remboursements ne se feront que sur les dons et les commandes non abouties.
Les paiements se feront en deux fois, un premier paiement de 25% avant que la commande ne soit commencée et une seconde fois une fois que la commande est récupérée.
Ils se feront sur ce compte [ici]

Comment récuperer sa commande ?

Je vous enverrai le fichier de votre commande sur Discord en message privé sous le format .rar (pour télécharger un décompresseur [ici]).
Dans le fichier .rar il y sera indiqué les modules nécessaires pour le bon fonctionnement du bot.
Si c'est une commande payante ou qu'un don a été effectuer la commande ne sera pas posté sur Github, dans le cas échéant seul vous aurez le fichier de votre bot.
