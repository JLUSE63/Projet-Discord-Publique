Comment faire une commande ?

C'est très simple il suffit de faire un devis en pdf, ce devis doit contenir:

  • Le type de bot: modération, statistique ou autre
  • Sur combien de serveur pourrait être le bot (un ou plus)
  • Le détail des commandes qu'il devra effectuer
  • Le token du bot (je ne prends pas le risque de prendre mon propre bot pour des problème de permissions)
  • Une invitation vers un serveur où je pourrais contacter quelqu'un en cas de question sur ce dernier (pour éviter de perdre du temps et avoir des réponse trop lente)
  • Votre désir d'un serveur tout fait pour vous avec un bot personnalisé [Payant]
  • Votre budget (merci de le préciser même si il est nul)

Une fois le devis prèt veuillez me l'envoyer sur mon serveur [ici] dans un ticket. Après la lecture de votre devis, je vous enverai un message pour vous dire si j'accepte ou non de faire votre commande. Je me donne le droit de refuser toute commande que je juge trop abusive ou ne respectant pas les ToS Discord

Comment payer ou faire un don ?

Tout payement ou don se fera via PayPal.
Je vous conseil d'utiliser un compte personnel et non professionnel lors de payement, dans le cas échéant si un remboursement doit être effectué vous ne pourrez avoir la totalité de votre payement/don.
Les remboursement ne se feront que sur les dons et les commandes non abouties.
Les payements se feront en deux fois, un premier payement de 25% avant que la commande ne soit commancée et une seconde fois une fois que la commande est récupérée.
Ils se feront sur ce compte [ici]

Comment récupperer sa commande ?

Je vous enverai le fichier de votre commande sur Discord en message privé sous le format .rar (pour télécharger un décompresseur [ici]).
Dans le fichier .rar il y sera indiqué les modules nécessaire pour le bon fonctionnement du bot.
Si c'est une commande payante ou qu'un don a été effectuer la commande ne sera pas posté sur Github, dans le cas échéant seul vous aurez le fichier de votre bot.
